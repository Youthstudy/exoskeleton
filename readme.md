# 下肢外骨骼控制系统

----

一个基于STM32F427II的下肢外骨骼控制系统，主要作用为辅助用户进行上台阶动作。

## 关于本项目

----

本项目旨在开发一款适合于人体下肢运动的外骨骼控制系统，在有限的资源下完成基于人体的姿态识别，上楼梯时的助力控制以及无助力状态时，电机自由地跟随人体运动。

## 环境要求

----

1. 软件要求

   - Keil 5

   - STM32CubeMx 6.9.2

2. 硬件要求

   - 下肢外骨骼相关设备
   - IMU
